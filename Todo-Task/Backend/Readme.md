
# Backend-ziptrip



API Endpoints

```
GET /api/todos: Fetch all todos
```

```
GET /api/todos/:id: Fetch a single todo by ID
```

```
POST /api/todos: Create a new todo
```

```
PUT /api/todos/:id: Update a todo by ID
```

```
DELETE /api/todos/:id: Delete a todo by ID
```
