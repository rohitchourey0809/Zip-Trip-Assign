# Backend-ziptrip



API Endpoints

```
GET /todos: Fetch all todos
```

```
GET /todos/:id: Fetch a single todo by ID
```

```
POST /todos: Create a new todo
```

```
PUT /todos/:id: Update a todo by ID
```

```
DELETE /todos/:id: Delete a todo by ID
```
